配置路径

  • 用户级别配置文件：~/.hcode/settings.json
  • 项目级别配置文件：项目根目录下的 .hcode/settings.json

配置项说明

属性名称 作用 是否支持热加载
env 环境变量配置，用于设置系统级别的环境变量 是
env.HAWA_CODE_LAZY_TOOL 工具懒加载开关 是
env.HAWA_CODE_FIX_SCHEMA 工具调用入参的兼容性修复选项 是
permissions.allow 权限配置，定义允许调用的工具（一般情况下无需手动修改） 是
language 界面语言设置，配置 Hawa Code 的默认回复语言 是
