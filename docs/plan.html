Plan

当对项目进行较大功能改动的时候，可以使用 hawa code 的 plan 模式。在 plan 模式下，hawa code 不会直接修改代码，而是先给出一个详细的修改方案，用户确认后再进行代码的修改，

进入 plan 模式，

  • 快捷键 Alt+m 切换到 plan 模式
  • 使用 command 命令 /plan 进入 plan 模式
  • 直接使用 进入 plan 模式 指令进入 plan 模式。
