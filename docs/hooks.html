Stop

在主 Agent 对话结束时触发，可用于阻止主 Agent 结束并让其继续工作。

入参

参数名 说明
session_id 当前会话 ID
transcript_path 当前会话的消息记录文件路径
permission_mode 当前 Agent 运行模式
hook_event_name Hook 事件名称
stop_hook_active 是否已触发过停止 Hook，true 表示已触发，false 表示未触发，用于避免无限循环
