Tasks

Hawa Code 任务管理已从 TodoWrite 升级为 Task 模式。

特性

Task 任务模式将任务列表存储在本地文件中，支持以下功能：

  • 跨 Subagent 协作：不同 Subagent 之间可以共享和同步任务状态
  • 跨会话持久化：通过配置环境变量，实现会话间的任务管理
  • 任务依赖管理：支持设置任务之间的依赖关系，确保按正确顺序执行

建议在执行大型工程项目时开启此模式，以获得更好的任务追踪和管理体验。
