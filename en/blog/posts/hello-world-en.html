Hawa Code 1.0 Officially Released

2024-01-15 Dahawa Team

We are excited to announce the official release of Hawa Code 1.0! This is an intelligent programming assistant designed for developers, aiming to improve coding efficiency and development experience through AI technology.

What is Hawa Code?

Hawa Code is an intelligent programming assistant based on large language models. It can understand developers’ intentions and provide code completion, error diagnosis, code refactoring, and many other features. Whether you are a front-end developer, back-end engineer, or full-stack developer, Hawa Code can be a valuable partner in your daily development work.

Core Philosophy

Our core philosophy is “an intelligent programming assistant who understands you.” We believe that good tools should understand developers’ needs, rather than forcing developers to adapt to the tools.

Use Cases

  • Code completion and suggestions in daily coding
  • Code review and quality improvement
  • Learning new technologies and frameworks
  • Rapid prototype development
  • Legacy code maintenance and refactoring

Key Features

Intelligent Code Completion

Hawa Code provides industry-leading code completion capabilities, supporting multiple programming languages and frameworks.

Supported Programming Languages

  • JavaScript/TypeScript: Complete syntax support and type inference
  • Python: Including data science and machine learning libraries
  • Java: Enterprise-level development support
  • Go: Cloud-native development preferred
  • Rust: System-level programming support
  • C/C++: Low-level development support

Framework Support

  • Front-end frameworks like React, Vue, Angular
  • Back-end frameworks like Django, Flask, Spring
  • Machine learning frameworks like TensorFlow, PyTorch

Multi-language Support

Hawa Code natively supports bilingual interface in Chinese and English, providing a great experience for developers from different language backgrounds.

Interface Languages

  • Chinese interface (default)
  • English interface
  • Automatic language detection

Code Comment Understanding

  • Supports Chinese and English code comments
  • Can understand intentions in comments
  • Generate corresponding code based on comments

Integrated Development Environment

Hawa Code deeply integrates with mainstream development environments, providing a seamless development experience.

IDE Support

  • VS Code: Official extension with the most complete features
  • IntelliJ IDEA: Including WebStorm, PyCharm, etc.
  • Vim/Neovim: Supported via LSP protocol
  • Emacs: Supported via LSP protocol

Terminal Support

  • Standalone terminal application
  • Supports mainstream shells (Bash, Zsh, Fish)
  • Compatible with Tmux, Screen, and other terminal multiplexers

Advanced Features

Agent Team

Supports multi-Agent collaboration, where complex tasks can be decomposed and handled by multiple specialized Agents.

Use Cases

  • Large project analysis
  • Cross-file refactoring
  • Code review
  • Documentation generation

Memory System

Hawa Code supports session memory and long-term memory, remembering your coding habits and project context.

Memory Types

  • Session Memory: Context of the current conversation
  • Project Memory: Knowledge specific to a project
  • User Memory: Personal coding preferences

Tool Integration

Supports integration with various tools:

  • File read and write operations
  • Code search and replace
  • Git operations
  • Test execution
